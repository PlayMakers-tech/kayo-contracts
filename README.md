# KAYO Work In Progress Smart Contracts

This repository is a testing grounds for the development of the blockchain behind KAYO (https://kayo.game)

Try running some of the following tasks:

```shell
npm install
npx hardhat test
npx hardhat run scripts/deploy.js
```

This is not supposed to be taken over at this point.

Documentation to be updated once all the interfaces are in good shape, and code properly commented.
